so let’s imagine we have some content that we’d like to put on the web. This could be anything, from simple pieces of text, graphics and images, lists, tables, all the way through to complex interactive content.
… we want our browser to be able to understand what that content is, so that it knows how to structure the webpage, and how to display things on the screen. This process, of taking our content and displaying it on the screen is known as rendering.
We use HTML tags to markup our content so that the browser will understand what our content is, and how to render it. there are many different tags in HTML, which can be used to add meaning and structure to different types of content, and we’ll look at some of these in later videos
Putting it all together, we have an opening ‘a’ tag and a closing ‘a’ tag, which together make up our anchor element. Our opening tag has an attribute which has a name and a value, and the element encloses some content. This is everything that goes in to making a basic HTML element.

# Block and Inline
### **Block** and Inline Some HTML tags produce **block elements** * **start** and **end** with a **new line** * **stack** together * take up **all** available **width** * can contain both **block** and **inline** elements * are used for **larger** pieces of **content**
### Block and **Inline** Some HTML tags produce **inline elements** * start without a **new line** * **line up** next to each other * as **wide** as their **content** * can contain other **inline** elements * **cannot contain** other **block elements** * used for **smaller** pieces of **content**
### Block and Inline * We can change how elements behave using CSS * Also have further options beyond 'block' and 'inline'
# Coding HTML
### Commenting We can add comments to our HTML code Comments will not be displayed to the user
### HTML Formatting Some HTML tags can add extra meaning to our text These tags may add additional formatting
### Special Characters Some characters in HTML are ‘reserved’ characters These must be replaced by ‘HTML entities’ to use them in a webpage
# Summary * HTML is used to annotate our content for the web * Use HTML tags to create HTML elements * Elements add meaning to our content * Nest elements inside one another to add structure to our document * Different elements have different functionality (and appearance)
